Variational message passingÂ¶

The general update equation for factorized approximation:

(1)\[\begin{split}\log q(\boldsymbol{\theta}) &= \langle \log p\left( \boldsymbol{\theta} | \mathrm{pa}(\boldsymbol{\theta}) \right) \rangle + \sum_{\mathbf{x} \in \mathrm{ch}(\boldsymbol{\theta})} \langle \log p(\mathbf{x}|\mathrm{pa}(\mathbf{x})) \rangle + \mathrm{const},\end{split}\]

where \(\mathrm{pa}(\boldsymbol{\theta})\) and \(\mathrm{ch}(\boldsymbol{\theta})\) are the set of parents and children of \(\boldsymbol{\theta}\), respectively. The expectations are over the approximate distribution of all other variables than \(\boldsymbol{\theta}\). Actually, not all the variables are needed, because the non-constant part uses only the Markov blanket of \(\boldsymbol{\theta}\). Thus, the optimization can be done locally using messages from neighbouring nodes.

The messages are simple for conjugate-exponential models. Exponential-family distributions have the form

(2)\[\begin{split}\log p(\mathbf{x}|\mathbf{\Theta}) &= \mathbf{u}_{\mathbf{x}}(\mathbf{x})^{\mathrm{T}} \boldsymbol{\phi}_{\mathbf{x}}(\mathbf{\Theta}) + g_{\mathbf{x}}(\mathbf{\Theta}) + f_{\mathbf{x}}(\mathbf{x}),\end{split}\]

where \(\mathbf{\Theta}=\{\boldsymbol{\theta}_j\}\) is the set of parents. If a parent has a conjugate prior, (2) is linear with respect to the parent’s natural statistics. Thus, (2) can be re-organized with respect to \(\boldsymbol{\theta}_j\) as

\[\begin{split}\log p(\mathbf{x}|\mathbf{\Theta}) &= \mathbf{u}_{\boldsymbol{\theta}_j}(\boldsymbol{\theta}_j)^{\mathrm{T}} \boldsymbol{\phi}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} (\mathbf{x}, \{\boldsymbol{\theta}_k\}_{k\neq j}) + \mathrm{const},\end{split}\]

where \(\mathbf{u}_{\boldsymbol{\theta}_j}\) is the natural statistics of \(\boldsymbol{\theta}_j\). Thus, the update equation (1) can be given as

\[\begin{split}\log q(\boldsymbol{\theta}_j) &= \mathbf{u}_{\boldsymbol{\theta}_j}(\boldsymbol{\theta}_j)^{\mathrm{T}} \left( \langle \boldsymbol{\phi}_{\boldsymbol{\theta}_j} \rangle + \sum_{\mathbf{x} \in \mathrm{ch}(\boldsymbol{\theta}_j)} \langle \boldsymbol{\phi}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} \rangle \right) + f_{\boldsymbol{\theta}_j}(\boldsymbol{\theta}_j) + \mathrm{const},\end{split}\]

where the summation is over all the child nodes of \(\boldsymbol{\theta}_j\). Because of the conjugacy, \(\langle\boldsymbol{\phi}_{\boldsymbol{\theta}_j}\rangle\) depends (multi)linearly on the expectations of the parents’ natural statistics. Similarly, \(\langle \boldsymbol{\phi}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} \rangle\) depends (multi)linearly on the expectations of the children’s and co-parents’ natural statistics.

The required expectations can be computed locally by using messages from the parents and the children. The message from a parent node \(\boldsymbol{\theta}_j\) to a child node \(\mathbf{x}\) is

\[\begin{split}\mathbf{m}_{\mathbf{\boldsymbol{\theta}_j}\rightarrow\mathbf{x}} &= \langle \mathbf{u}_{\boldsymbol{\theta}_j} \rangle = \tilde{\mathbf{u}}_{\boldsymbol{\theta}_j} (\tilde{\boldsymbol{\phi}}_{\mathbf{\boldsymbol{\theta}_j}}),\end{split}\]

and the message from a child node \(\mathbf{x}\) to a parent node \(\boldsymbol{\theta}_j\) is

\[\begin{split}\mathbf{m}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} &= \langle \boldsymbol{\phi}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} \rangle = \boldsymbol{\phi}_{\mathbf{x}\rightarrow\boldsymbol{\theta}_j} \left( \langle \mathbf{u}_{\mathbf{x}} \rangle, \{ \mathbf{m}_{\theta_k\rightarrow\mathbf{x}} \}_ %\{ \langle \mathbf{u}_{\theta_k} \rangle \}_ {k \neq j} %\in \mathrm{cp}({\boldsymbol{\theta}_j}, \mathbf{x})} \right).\end{split}\]

Using the messages, the natural parameters of \(q(\boldsymbol{\theta})\) can be updated as

\[\begin{split}\tilde{\boldsymbol{\phi}}_{\boldsymbol{\theta}} &= \boldsymbol{\phi}_{\boldsymbol{\theta}} \left( \{ \mathbf{m}_{\mathbf{z}\rightarrow\boldsymbol{\theta}} \}_ {\mathbf{z} \in \mathrm{pa}(\boldsymbol{\theta})} \right) + \sum_{\mathbf{x} \in \mathrm{ch}(\boldsymbol{\theta})} \mathbf{m}_{\mathbf{x}\rightarrow\boldsymbol{\theta}}.\end{split}\]
