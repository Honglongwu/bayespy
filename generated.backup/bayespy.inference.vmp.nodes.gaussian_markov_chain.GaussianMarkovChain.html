bayespy.inference.vmp.nodes.gaussian_markov_chain.GaussianMarkovChain¶

class bayespy.inference.vmp.nodes.gaussian_markov_chain.GaussianMarkovChain(*args, **kwargs)[source]¶

VMP node for Gaussian Markov chain.

Parents are: mu is the mean of x0 (Gaussian) Lambda is the precision of x0 (Wishart) A is the dynamic matrix (Gaussian) v is the diagonal precision of the innovation (Gamma) An additional dummy parent is created: ‘N’ is the number of time instances

Output is Gaussian variables.

Time dimension is over the last plate.

Hmm.. The number of time instances is one more than the plates in A and V. Input N -> Output N+1.
